## Contributing to docs

Community contributions are welcome. Statsig's docs run on [Mintlify](https://www.mintlify.com/). Open a PR and request a review in our [Slack channel](https://statsig.com/slack) to merge your change in.

### Local development:

```
npm i -g mintlify
mintlify dev
```

#### Troubleshooting

- Mintlify dev isn't running - Run `mintlify install` it'll re-install dependencies.
- Page loads as a 404 - Make sure you are running in a folder with `docs.json`
